## 💡 서비스 소개
- 서비스명 : SJG
- 판매상품 : 타일(Porcelain, Walltile, Floortile)
- 고객 : 인테리어에 관심이 많은 개인 구매자, 대리구매를 하는 타일 사업자
- 컨셉
  - 이솝 사이트를 기본 베이스를 하여 친환경에 초첨을 두며 만들었습니다.
  - 이솝 사이트의 레이아웃을 토대로 최대한 비슷하게 만들었으며
실제 사이트를 사용했을때 불편한점을 저희가 기능을 추가하여 사용자의 편의성을 상승시켰습니다.
